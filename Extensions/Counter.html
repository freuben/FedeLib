Counter a counter


Counter.ar(highLimit, lowLimit, whichDir)


This should say what the class does and how great it is. 


highLimit - sets higher limit

lowLimit - sets lower limit

whichDir - set which direction to count


A simple counter.


Example


c = Counter(10);


c.step; //counts each time evaluated


//set high and low limits:

c.highLimit = 15; 

c.lowLimit = -4;


c.direction(\down); //counts backwards


c.step; 


c.direction(\up); //back to normal


c.step; 



c.function = {"bang".postln}; //evaluate this function everytime the counter reaches the high limit


c.highLimit = 10; 

c.lowLimit = 0;


c.count = 10;


{inf.do{c.step.postln; 0.3.yield}}.fork; //evaluate in a routine


c.direction(\upDown); //counts up and down


c.whichDir; //tells you which direction mode


c.carry; //count loops



//multiple Counters:


//defaul argument for highLimit is inf

~array = Counter.new!10; //an array of 10 counters

~count = Counter(~array.size-1);//a counter to get indexes


(

{inf.do{

var countArr;

~array.do({|item| countArr = countArr.add(item.count)});

countArr.postln;

~array[~count.step].step;

0.05.yield;

}}.fork;
