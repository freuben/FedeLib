Spectral Tracking:


PartialTracker( windowSize,  numberPartials,  fftbuffer, magbuffer,  freqbuffer)


Uses Josh's PV_MagBuffer and PV_FreqBuffer to get FFT information of an audio input into an Array. You can specify the amount of partials you want from maximum to minimum.


an example:


p = PartialTracker(512, 5, 1, 2, 3); 

p.startsynth(1); //use AudioIn input - channel 1

p.getarrays;

p.frequencies; //get array with frequencies

p.magnitudes; //get array with magnitudes



FFTFilter( windowSize,  numberPartials,  fftbuffer, magbuffer,  freqbuffer)

Bandpass filter that takes FFT info from vocal input (1) to shape a sound from the input of a buffer or live input (5).


s.sendMsg(\b_allocRead, 3, "sounds/a11wlk01.wav");


p = FFTFilter(256, 255, 0, 1, 2); 

p.startsynth3buf(1,0,2.0,3); //vocalIn, out, volume, soundbuffer


p.getarrays;

p.thresh(2.4);

p.getQu;


p.volume(2.0);


p.stoptask;

p.resume;


SpearToSC( pathName )


Takes SPEAR (by Michael Klingbeil) text files saved in export format as paritals - and stores them into a SuperCollider Array.


a = SpearToSC("/Users/freuben/Music/E-tudes/Analized Stuff/Recorder/prueba.txt"); //your own path

a.getData //this might take some time, wait until it says "done"

g = a.dataArray; //all partials in multidimentional array

g.dopostln; //post them

g.size //amount of partials


g[0].postln; //get first partial

g[0].flop[0] //times

g[0].flop[1] //frequencies

g[0].flop[2] //amplitudes


SpearToMIDI( pathName )


Takes SPEAR (by Michael Klingbeil) text files - paritals - and reduces data to write a MIDI File with selected information. You can specify if you want note or gliss notation and track division in semitones.


a = SpearToMIDI("/Users/freuben/Documents/SuperCollider/Fede's/Aleks/other/radiohead.txt") //your own path

a.getData //this might take some time, wait until it says "done"


a.dataArray.size //number of partials


//reduces data to notes notation

a.getDataNote(1.0, 0.1); //this might take some time, wait until it says "done". arguments are: Midi modulation and amplitude threshold


a.dataNote.size;


a.midiFileNote("~/Desktop/radiohead.mid", 120, "4/4", 6, 0); //creates midi file, this takes some time. arguments are: pathName for new midiFile, tempo, time signature, track division in semitones, filternotes out of range (if arg == 1)


Real-Time Scoring:


AlgorithnicScore( size )


A class for scoring music in real-time:


a = AlgorithmicScore(1.2);


a.score([\piano], 1.0, 1, 1.3); //clefArray, zoom, down, staffGap, spacing


(

~notes = (60,61..76);

~pos = (0,1..16);

a.notes(~notes, ~pos);

)


a.notes(~notes-7, ~pos);


a.close;

t.getTime; //get's time, try it several times


t.reset; //takes the stopwatch back to cero

Recording:


A class to record to a file with a given path:


RecordToFile(pathName, numChannels,sampleFormat)


pathName - the path you want your file to be saved.

numChannels - number of channels you want to record in the File.  

sampleFormat   - the sample format of you file:

16 integer: "int16"

24 integer:  "int24"

etc...

r = RecordToFile("sounds/test.aif", 1, "int24");


r.startRec; //start recording


a = {SinOsc.ar(400, 0, 0.1)}.play(s) //something to record


r.stopRec; //stop recording
